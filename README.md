# 基于深度学习的遥感图像水体提取系统

本项目是一个集成了改进 **U-Net** 模型与响应式 Web 前端交互的遥感影像处理平台 。系统实现了从超大遥感图像上传、自动化水体检测到多维度结果可视化的全流程功能 。

## 核心特性

### 1. 高性能前端架构

**大数据量处理**：针对超过 **1GB** 的大尺寸遥感图像，实现了 **4MB 分片上传**、**MD5 校验**及**断点续传**功能，确保了上传过程的稳定性 。

**主线程优化**：利用 **WebWorkers** 异步生成图像缩略图，有效避免了高分辨率图像处理导致的主线程阻塞 。

**交互式对比**：开发了基于 **Canvas** 的滑动对比查看器，支持检测结果与原图的实时像素级变化对照 。



### 2. 精准的水体识别算法


**模型架构**：采用改进的 **U-Net** 深度学习模型进行像素级水体分类 。



**性能指标**：模型在水体提取任务中达到了 **0.892 的 IoU** 指标 。



### 3. 多维度可视化交互


**GIS 集成**：使用 **Leaflet.js** 开发交互地图模块，支持 SVG 区域框选、多图层叠加以及检测结果的实时渲染 。



**数据看板**：集成 **ECharts** 与 **D3.js**，构建了包括覆盖率饼图、面积变化折线图及热力图在内的多视图组件 。

## 项目展示

### 图像处理流程

1. **分片上传**：前端将大图切分为 4MB 的数据分片，结合 MD5 校验确保文件完整性 。
2. **异步预处理**：WebWorkers 处理图像缩放与预览逻辑 。
3. **模型推理**：后端调用 U-Net 模型进行识别。
4. **实时反馈**：检测结果通过 Canvas 滑动对比器进行展示 。



### 统计分析

* 系统支持一键导出可视化报表，涵盖提取面积统计与覆盖率分析结果 。
